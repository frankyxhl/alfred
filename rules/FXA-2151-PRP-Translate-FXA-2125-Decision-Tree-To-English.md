# PRP-2151: Translate FXA-2125 Decision Tree To English

**Applies to:** FXA project
**Last updated:** 2026-03-30
**Last reviewed:** 2026-03-30
**Status:** Implemented

---

## What Is It?

Proposal to translate the Chinese-language Project Decision Tree in FXA-2125 to English, resolving a COR-0002/COR-1401 compliance drift.

---

## Problem

FXA-2125 (Workflow Routing PRJ) contains its Project Decision Tree section entirely in Chinese. COR-0002 states: "All documents must be written in English. See COR-1401." COR-1401 Policy §1 states: "All documents must be written in English."

This is a format contract violation discovered during Evolve-SOP content analysis (FXA-2150 run log).

## Proposed Solution

Translate the 9 decision-tree entries from Chinese to English while preserving the structure, SOP references, and command examples. No content additions or removals — pure translation.

## Open Questions

None — the policy is unambiguous and the translation is mechanical.
